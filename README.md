# IT1214 - Student Grade Management System

### Project Purpose
This is a Java console-based application developed to digitize and manage academic records for a private educational institute. It replaces manual paper and spreadsheet records, allowing teachers to store student details, calculate performance averages, and search records efficiently using OOP concepts.

### Features
1. Add new student details (ID, Name, Marks).
2. Display all existing student records.
3. Search for a specific student using their Student ID.
4. Calculate and display the average marks of all students.
